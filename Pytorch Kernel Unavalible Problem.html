pytorch 报错 RuntimeError: CUDA error: no kernel image is available for execution on the device

环境：

  • python 3.8.3
  • pytorch 1.7.1
  • cuda10.1
  • GPU: GeForce 920M

问题

用GPU运行代码时报错如下

因为我的cuda是配好了的，用tensorflow时就没问题，本来运行以下测试的时候发现也是没问题的。

In [2]: torch.cuda.is_available()
Out[2]: True
