Python 写一个下载文件的脚本，“http://SOME_DOMAIN.XX.pdf”，其中，XX需要填上00到99的数字（下载时需要带进度条）:
```python
import requests
from tqdm import tqdm

base_url = "https://www.marxists.org/chinese/pdf/marx-engels/me{:02d}.pdf"


for i in range(100):
    url = base_url.format(i)
    response = requests.get(url, stream=True)
    
    if response.status_code == 200:
        file_name = f"file_{i:02d}.pdf"
        total_size = int(response.headers.get("content-length", 0))
        
        with open(file_name, "wb") as file, tqdm(
            desc=file_name,
            total=total_size,
            unit="iB",
            unit_scale=True,
            unit_divisor=1024,
        ) as progress_bar:
            for data in response.iter_content(chunk_size=1024):
                file.write(data)
                progress_bar.update(len(data))
        
        print(f"文件 {file_name} 下载成功！")
    else:
        print(f"链接 {url} 无法访问或文件不存在。")
```
